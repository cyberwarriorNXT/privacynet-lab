# PrivacyNet Lab 🔐

An educational, sandboxed multi-hop encrypted routing network.
Built to make privacy networking concepts visible and understandable.

## What This Project Does
Routes traffic through multiple encrypted hops so that no single 
node knows both the origin and destination of the traffic.

## Current Status
🔲 Phase 1 — Two-node encrypted communication (in progress)

## Roadmap
- [ ] Two-node socket connection
- [ ] AES-256 layered encryption  
- [ ] Three-hop circuit
- [ ] Terminal logs per node
